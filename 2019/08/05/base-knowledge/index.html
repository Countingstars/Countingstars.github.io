逻辑与「&&」

逻辑与可以操作任何类型的操作数。逻辑与是短路操作，如果第一个操作数能决定结果，则不会对第二个操作数求值

第一个操作数 第二个操作数 结果
true true true
true false false
false true false
false false false
操作数 – 返回第二个操作数
true 操作数 返回第二个操作数
false – 返回false
null/NaN/undefined null/NaN/undefined 其中一个为null/NaN/undefined则返回null/NaN/undefined
